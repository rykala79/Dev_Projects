About Me

Hi, I'm Tomasz! I'm a Frontend Developer with a background in electronics and a passion for web development.
As a certified electronics technician, I have a strong technical foundation that helps me understand both hardware and software aspects of IT.
I specialize in HTML, CSS, and JavaScript, and I work with modern JavaScript frameworks such as React and Next.js to build dynamic and responsive web applications.
I’m passionate about creating clean, user-friendly interfaces and continuously improving my development skills.
I’m also currently learning IT support, expanding my knowledge in troubleshooting,
system maintenance, and user support.
In addition to tech, I love learning new languages and improving my German. I enjoy working on creative projects, including developing my personal website and exploring new technologies.
